# Marketing — El Maravilloso

Sistema de creación de contenido para redes sociales (Instagram + TikTok) de El Maravilloso (distribuidora comercial).

## Estructura
- `brand/brand.md` — Identidad de marca: voz, colores, tono, público. **Llenar primero.**
- `assets/` — Logos, fotos de producto, recursos visuales reutilizables.
- `templates/` — Plantillas HTML para generar imágenes/social photos (HTML → screenshot).
- `content/instagram/` — Posts generados para Instagram.
- `content/tiktok/` — Guiones / contenido para TikTok.
- `calendar.md` — Calendario y backlog de publicaciones.

## Flujo de trabajo
1. Completar `brand/brand.md` (una vez).
2. Pedir contenido → las skills de diseño (`design`, `brand`, `canvas-design`) leen la marca y generan los assets.
3. Revisar en `content/` y publicar (manual o vía API según se defina).
